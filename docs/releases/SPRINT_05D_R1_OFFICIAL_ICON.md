# Aetheris Sprint 05D-r1 — Official Launcher Icon

## Objetivo

Substituir o ícone vetorial experimental da Sprint 05D pelo ícone oficial escolhido pelo usuário.

## Alterações

- Ícone oficial do Aetheris aplicado a partir de arquivo local da pasta Pictures.
- PNGs legacy gerados para mdpi, hdpi, xhdpi, xxhdpi e xxxhdpi.
- Foreground adaptativo gerado para Android moderno.
- Manifest mantido usando `@mipmap/ic_launcher` e `@mipmap/ic_launcher_round`.
- Removido foreground vetorial experimental.
- Removido monochrome experimental.
- Mantido `ic_launcher_background` apenas em `colors.xml`.
- Versão atualizada para `0.2.3-m3icon-r1`.
